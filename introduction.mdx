## Welcome to CoW Protocol BFF

CoW Protocol BFF (Backend For Frontend) is a comprehensive collection of TypeScript-based backend services designed to power the CoW Swap frontend and other CoW Protocol applications. Built with modern tools and best practices, it provides essential services for token pricing, notifications, yield aggregation, and more.

## Key Features

<AccordionGroup>
  <Accordion title="Multi-Chain Token Services" icon="coins">

## Technology Stack

The BFF is built on a modern TypeScript stack:

- **Framework**: Fastify for high-performance HTTP services
- **Monorepo**: NX for efficient workspace management
- **Database**: PostgreSQL with TypeORM
- **Caching**: Redis with multi-layer fallback strategies
- **Messaging**: RabbitMQ for asynchronous notifications
- **Blockchain**: Ethers.js and Viem for web3 interactions

- **Services**: Business logic for token prices, balances, simulations, etc.
- **Shared**: Common utilities, types, and constants
