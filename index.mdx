  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get the BFF services running locally in minutes
  </p>

  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install dependencies with Yarn.
      
      ```bash
      git clone https://github.com/cowprotocol/bff.git
      cd bff
      yarn install
      ```
    </Step>
    
    <Step title="Configure environment">
      Create a `.env` file from the example template.
      
      ```bash
      cp .env.example .env
      ```
      
      Edit `.env` to add your RPC URLs and API keys for the chains you want to support.
    </Step>
    
    <Step title="Start services">
      Launch PostgreSQL, RabbitMQ, and Redis with Docker Compose, then start the API.
      
      ```bash
      docker-compose up -d db queue redis
      yarn migration:run
      yarn start
      ```
      
      The API will be available at `http://localhost:3001`
    </Step>
    
    <Step title="Test the API">
      Make your first API call to get token prices.
      
      ```bash
      curl http://localhost:3001/1/tokens/0xC02aaA39b223FE8D0A0e5C4F27eAD9083C756Cc2/usdPrice
      ```
      
      <Accordion title="Example response">
        ```json
        {
          "price": 3561.1267842
        }
        ```
      </Accordion>
    </Step>
  </Steps>
</div>
